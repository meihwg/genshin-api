# Genshin API

## Description

A public GitHub raw API for Genshin Impact.
Because of the API being raw you can't apply filters on the request, you need to retrive all data and filter on your code.

## Paths

Use this link for every request and replace [filepath] with the path of the file you want in the GitHub repository :

https://raw.githubusercontent.com/meihwg/genshin-api/main/[filepath]

All usable links :

https://raw.githubusercontent.com/meihwg/genshin-api/main/game-data.json

https://raw.githubusercontent.com/meihwg/genshin-api/main/characters.json
